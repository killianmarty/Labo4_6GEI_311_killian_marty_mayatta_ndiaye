 t.\+Optional\mbox{[}\mbox{\hyperlink{classflask_1_1sessions_1_1_secure_cookie_session}{Secure\+Cookie\+Session}}\mbox{]} flask.\+sessions.\+Secure\+Cookie\+Session\+Interface.\+open\+\_\+session (\begin{DoxyParamCaption}\item[{}]{self}{, }\item[{"{}Flask"{}}]{app}{, }\item[{"{}Request"{}     }]{request}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}This is called at the beginning of each request, after
pushing the request context, before matching the URL.

This must return an object which implements a dictionary-like
interface as well as the :class:`SessionMixin` interface.

This will return ``None`` to indicate that loading failed in
some way that is not immediately an error. The request
context will fall back to using :meth:`make_null_session`
in this case.
\end{DoxyVerb}
