str inflection.\+camelize (\begin{DoxyParamCaption}\item[{str}]{string}{, }\item[{bool }]{uppercase\+\_\+first\+\_\+letter}{ = {\ttfamily True}}\end{DoxyParamCaption})}

\begin{DoxyVerb}Convert strings to CamelCase.

Examples::

    >>> camelize("device_type")
    'DeviceType'
    >>> camelize("device_type", False)
    'deviceType'

:func:`camelize` can be thought of as a inverse of :func:`underscore`,
although there are some cases where that does not hold::

    >>> camelize(underscore("IOError"))
    'IoError'

:param uppercase_first_letter: if set to `True` :func:`camelize` converts
    strings to UpperCamelCase. If set to `False` :func:`camelize` produces
    lowerCamelCase. Defaults to `True`.
\end{DoxyVerb}

\begin{DoxyVerb}Make an underscored, lowercase form from the expression in the string.
